# Data Analytics Dashboard

## Overview
A Python-based Data Analytics Dashboard that processes, analyzes, and visualizes structured sales data.
The project demonstrates data cleaning, statistical analysis, SQL-based aggregation, and dashboard
visualization using Jupyter Notebook.

## Tech Stack
- Python
- Pandas
- NumPy
- SQLite
- Jupyter Notebook
- Matplotlib

## Features
- Cleaned and processed CSV data using Pandas
- Generated derived metrics such as total sales and order value
- Performed statistical analysis using NumPy
- Stored and queried data using SQLite
- Built a notebook-based analytics dashboard with visual insights
- Modular project structure for reusability and scalability
